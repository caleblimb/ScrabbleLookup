# ScrabbleLookup
Scrabble Word Utility

# Overview
This program is a simple utility for verifying Scrabble words.

# Development Environment
* Kotlin v1.6.21
* Android Studio Chipmunk | 2021.2.1
* Compile SDK 32
* JVM Target 1.8
